Git

In this class we are only going to be using vscode to commit to github.

The steps necessary are:

  • Make changes to code
  • Ensure the changes are saved
  • The src control icon on the left will show a number which indicates the number of files that have been changed
  • click on the src control icon
  • stage the files you want to commit by clicking the +
  • enter a commit message and click on "commit" to commit the files (these changes are only on your computer at this point)
  • click the "sync" button to move the files to github (if successful, you should see them in your account in github)

Barely more advanced

There are a lot of tools to help with git, but you can simply use the command line in a terminal

Below is as simple as I can make it, there is much, much more you can do!

  • git status - to show status of files
  • git add "filename" - to stage the file
  • git commit -m 'message' - commits staged files with commit message
  • git push - pushes the committed messages to your github account
  • git pull - grab the current state in github
